# Customer Shopping Data Analysis Project

**Customer Shopping Data Analysis using Python, MySQL, and Power BI**

## 📖 Project Overview

This project analyzes customer shopping behavior using a dataset containing **6,000 customer records**. The objective is to identify purchasing trends, customer demographics, seasonal sales patterns, and business insights that can help improve sales and customer satisfaction.

The project follows a complete data analytics workflow:

* Data Collection
* Data Cleaning
* Data Analysis using SQL
* Data Visualization in Power BI
* Business Recommendations

---

## 🎯 Objectives

* Analyze customer purchasing behavior.
* Identify top-selling product categories.
* Compare spending based on gender and age groups.
* Analyze seasonal sales trends.
* Study subscription and discount impacts on revenue.
* Build an interactive Power BI dashboard for business decision-making.

---

## 🛠️ Tools & Technologies

* **Python (Pandas)** – Data Cleaning & Preprocessing
* **MySQL Workbench** – Data Storage & SQL Queries
* **Power BI** – Interactive Dashboard
## 📂 Dataset Information

The dataset contains **6,000 customer shopping records** with the following columns:

* Customer ID
* Age
* Gender
* Item Purchased
* Category
* Purchase Amount (Rs.)
* Location
* Size
* Color
* Season
* Review Rating
* Subscription Status
* Shipping Type
* Discount Applied
* Promo Code Used
* Previous Purchases
* Payment Method
* Frequency of Purchases

## 📊 Key Insights

* Male and female customers contribute significantly to total revenue.
* Clothing and accessories are among the highest-selling categories.
* Seasonal shopping patterns influence customer purchases.
* Discounts and promo codes increase purchase frequency.
* Subscription members tend to make more repeat purchases.
* Certain payment methods are preferred by customers.

---

## 💼 Business Recommendations

* Increase promotions during peak shopping seasons.
* Offer personalized discounts based on customer purchase history.
* Strengthen loyalty programs for subscription members.
* Focus marketing efforts on high-performing product categories.
* Expand inventory for frequently purchased products.
* Improve customer retention using targeted promotional campaigns
